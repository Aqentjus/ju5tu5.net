  1. go to the directory on your computer where your hugo site files are located.

  2. Cmd on that directory and use the command:

hugo -D

  1. get you public folder and its files to github

  2. Use PUTTY to access your server by typing in the ip address of your server.

  3. Next run these commands to make sure that the software is uptodate:

sudo apt update

sudo apt upgrade
  1. Next install apache2, git and PHP by running these commands
sudo apt install apache2 -y

sudo apt install git -y

sudo apt install php -y

  1. After this you can go to your server ip address on your browser and you should see this:

  2. Now navigate to the correct directory by using this command:

cd /var/www/html
  1. After that you can run this command to clear the directory:
sudo rm *
  1. Now we want to get the files from github by using this command(you need to replace the link with your own repository link):
sudo git clone https://github.com/Aqentjus/ju5tu5.net.git
  1. Now you need to make it so the www directory eventually looks like this:

I achieved this using these commands so they might be helpful for you too:

ls -l

cd ju5tu5.net/public/

sudo cp * ../.. -R

cd ../..

ls -l

sudo rm ju5tu5.net/ -R
